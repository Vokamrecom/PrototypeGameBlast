# PrototypeGameBlast

Прототип игры-головоломки с механикой Blast на Cocos Creator 2.4.x

## Игровая механика

- **Поле**: 8×8 тайлов разных цветов
- **Цель**: Набрать 1000 очков за 30 ходов
- **Механика**: Клик по тайлу сжигает все прилегающие тайлы того же цвета (минимум 2)
- **Гравитация**: После сжигания тайлы падают вниз, пустые ячейки заполняются новыми
- **Очки**: Количество сожженных тайлов² × 10

## Старт

### Установка
1. Клонируйте репозиторий
2. Откройте проект в Cocos Creator
3. Откройте сцену `assets/scenes/MainScene.fire`
4. Нажмите **Запуск** 

## Технологии

- **TypeScript** - типизированный JavaScript
- **Cocos Creator 2.4.x** - игровой движок



##  Особенности

-  Алгоритм поиска связанных тайлов (BFS)
-  Механика гравитации
-  Генерация новых тайлов
-  Система подсчета очков
-  Проверка условий победы/поражения
-  Анимации сжигания и падения
-  Адаптивный UI для мобильных устройств

##  Структура проекта

```
assets/
  scripts/
    models/        - Модели данных
    controllers/    - Контроллеры
    views/          - Представления (UI)
    managers/       - Менеджеры игровой логики
    utils/          - Утилиты
  scenes/          - Сцены игры
  resources/       - Ресурсы (спрайты)
```
